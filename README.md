# Infinite Dogs Collage

### `Summary`

Project followed using a tutorial online (linked below) to get used to using fetch calls in vanilla Javascript to receive data and show the data on the browser. Users can choose with a menu selector to see which kind of dogs they want to see.

### `Languages / Techstack'

- HTML
- CSS
- JavaScript (Fetch)
